# 语义标签采用 PCIC 官方分类体系（Nociones Específicas）

- 日期：2026-07-10   状态：已采纳

## 背景

西语项目的语义标签一级大类原是**直接复用韩语姊妹项目**的「通用 20 大类」（饮食·食物·烹饪 / 情感·心理·性格 / 动作·行为·过程 / 状态·性质·变化…）。这套分类是为通用中文语义设计的，并非西班牙语教学的权威体系，存在问题：

- 用户质疑「有没有西班牙语/拉丁语等官方的分类体系」。
- 「动作·行为·过程」「状态·性质·变化」这类偏语法功能的大类，在词汇教学里不如主题场景分类实用。
- 缺乏可对外说明的权威来源。

## 抉择

- **选项 A**：保留通用 20 类，仅在文档/prompt 里标注 PCIC 作参考——改动小，但不算真正采用官方体系。
- **选项 B**：换成 **Instituto Cervantes《Plan Curricular》(PCIC) 的 Nociones Específicas** 官方 20 类。PCIC 是西语作为外语（ELE）教学的事实标准（对齐 CEFR/欧洲语言共同参考框架），其「具体概念」清单本就是按主题场景组织的词汇分类。
- **选项 C**：PCIC + 通用两套并存，标签可挂任一体系——最灵活但标签云臃肿、认知负担大。

## 决定

采用 **B**，换成 PCIC Nociones Específicas 官方 20 类（中文标签）：

个体·身体维度 / 个体·感知与心理 / 个人身份 / 人际关系 / 饮食 / 教育 / 工作 / 休闲娱乐 / 信息与媒体 / 居住 / 公共服务 / 购物与商店 / 健康与卫生 / 旅行·住宿·交通 / 经济与工业 / 科学与技术 / 政府·政治·社会 / 艺术活动 / 宗教与哲学 / 地理与自然。

- 来源：<https://cvc.cervantes.es/ensenanza/biblioteca_ele/plan_curricular/>（Niveles → Nociones específicas 清单）。
- `TAG_TAXONOMY_ROOTS` / `_TAG_SUB_EXAMPLES` / `_routeTagToRoot` 全部改为 PCIC。
- 历史数据迁移：`_OLD_TO_PCIC_ROOT` 旧→新映射 + `migrateTaxonomyToPCIC()`（幂等，`spanish_taxonomy_pcic_v1` 版本守卫），迁移标签图父链、词条 tags、悬空 parent 兜底。
- 种子数据、AI 新词条 prompt、AI 补全 prompt、AI 标签重组 prompt 全部对齐 PCIC。

## 后果

- 分类有权威出处，可对外说明，且与 CEFR/ELE 教学体系一致。
- 偏语法功能的旧大类（动作·行为·过程 / 状态·性质·变化 / 抽象概念）在 PCIC 里没有对应主题类，迁移时按语义并入「科学与技术 / 地理与自然」等，可能有个别语义漂移——可后续用 AI 标签重组微调。
- **与韩/日姊妹项目分歧**：韩/日仍用通用 20 类。这是西语**有意偏离**姊妹项目对齐原则的一处：因为西语有 PCIC 这样的官方权威，而韩/日项目暂无等价选择。若日后韩/日找到各自官方词汇分类（如日本語能力試験語彙表），应各自采用，而非强行统一。
- 旧标签作为叶子仍保留在标签图中（迁移只改父链，不删叶子），历史词条的具体标签不丢。
